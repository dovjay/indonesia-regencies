# indonesia-regencies

Geojson files of all regencies in Indonesia per provinces.

## Data Source

The geojson files in this repository are sourced from [Humanitarian Data Exchange (HDX)](https://data.humdata.org/dataset/cod-ab-idn).

**Source URL:** https://data.humdata.org/dataset/cod-ab-idn

## Structure

The repository contains geojson files organized by Indonesian provinces. Each file follows the naming convention:
- `kab37_geojson_[PROVINCE_NAME].json`

Where `[PROVINCE_NAME]` is the uppercase name of the province (e.g., `ACEH`, `JAWA_BARAT`, `DKI_JAKARTA`).
